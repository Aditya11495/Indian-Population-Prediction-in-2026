**Indian Population Prediction using Linear Regression**

**Project Overview**
This project predicts the population of India based on historical population data using Linear Regression, a supervised machine learning algorithm.
 The objective is to understand population growth trends and apply regression techniques for future prediction.
This project is created for learning machine learning fundamentals, especially regression analysis, data preprocessing, and model evaluation.

**Problem Statement**
To predict India's future population using historical population data by applying Linear Regression.

**Dataset**
Historical population data of India

Population
